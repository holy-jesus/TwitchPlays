# TwitchPlays Bot

Python-бот для создания интерактивных трансляций в стиле "Twitch Plays", где зрители могут управлять игровым процессом через команды в чате Twitch. 

Бот использует [twitchAPI](https://github.com/Teekeks/twitchAPI) для получения сообщений из чата и [ahk](https://github.com/spyoungtech/ahk) (AutoHotkey) для эмуляции нажатий клавиш и движений мыши в указанном окне игры.

## Особенности
* **Привязка к окну**: Команды из чата применяются только если активно указанное окно игры (например, `GTA5.exe` или `javaw.exe`).
* **Кулдауны**: Встроенная система задержек (cooldowns) предотвращает спам.
* **Голосования (Democracy Mode)**: Возможность настроить команды, которые сработают только если наберут нужное количество голосов за определенное время.
* **Плавная мышь**: Плавное движение камеры/мыши с настраиваемой скоростью.
* **Горячие клавиши**: Удобное управление самим ботом (пауза, экстренная остановка).
* **Кастомизация**: Поддержка алиасов (синонимов) для команд, например, русская и английская раскладки.

## Требования
* Python 3.10+
* Установленный AutoHotkey (AHK) v1.x (путь должен быть доступен в PATH).
* Зависимости Python: `twitchAPI`, `ahk`

## Быстрый старт
Пример настройки для Minecraft:
```python
from bot import Bot, Keys, Direction, RUSSIAN_WASD

# Инициализация бота для канала, процесс игры "javaw.exe"
bot = Bot("CHANNEL_NAME", "javaw.exe", mouse_speed=10)

# Регистрация команд передвижения (WASD)
bot.register_wasd(0.3, cooldown=0.3, **RUSSIAN_WASD)

# Запуск бота
bot.run()
```

## Горячие клавиши (на самом ПК)
* `F12` — Поставить бота на паузу / Снять с паузы. Зрители пишут команды, но они игнорируются.
* `Shift` + `Backspace` — Полностью остановить работу бота.
